### 2.build on Windows

#### dependency:

* Cmake GUI
* Visual Studio 2022

#### build:

* Open CMake GUI
* Add "SoundRender-main" as source directory. Add "/build" after this path as build directory
* Click "configure". Choose VS2022 as the generator 
* Click "generate". Open ".sln" file in the "build" directory.
